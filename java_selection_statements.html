🔵 if Операторы

if операторы — ең көп қолданылатын таңдау операторы

Жалпы синтаксис:

if (шарт) {
    // Шарт ақиқат болса орындалатын код
} else if (басқа_шарт) {
    // Басқа шарт ақиқат болса орындалатын код
} else {
    // Ешбір шарт ақиқат болмаса орындалатын код
}

💻 if Операторының Мысалдары

📝 Қарапайым if операторы
// 1. Қарапайым if int age = 20; if (age >= 18) { System.out.println("Сіз ересек адамсыз"); } // 2. if-else int temperature = 25; if (temperature > 30) { System.out.println("Ыстық"); } else { System.out.println("Жылы немесе суық"); } // 3. if-else if-else int score = 85; if (score >= 90) { System.out.println("Өте жақсы"); } else if (score >= 80) { System.out.println("Жақсы"); } else if (score >= 70) { System.out.println("Қанағаттанарлық"); } else { System.out.println("Нашар"); }

💡 if Операторының Ерекшеліктері:

• Шарт міндетті түрде boolean мәнін қайтаруы керек
• Жақшалар {} қолдану ұсынылады, тіпті бір жол код болса да
• else if блоктарының саны шектелмеген
• else блогы міндетті емес

🔗 Күрделі Шарттар

🧠 Логикалық операторлармен шарттар
// AND (&&) операторы int age = 25; boolean hasLicense = true; if (age >= 18 && hasLicense) { System.out.println("Көлік жүргізе алады"); } // OR (||) операторы String day = "сенбі"; if (day.equals("сенбі") || day.equals("жексенбі")) { System.out.println("Демалыс күні"); } // NOT (!) операторы boolean isRaining = false; if (!isRaining) { System.out.println("Серуенге шығуға болады"); } // Күрделі шарттар int temperature = 22; boolean isSunny = true; if (temperature > 20 && temperature < 30 && isSunny) { System.out.println("Керемет ауа-райы!"); }

⚠️ Жиі кездесетін қателер:

• if (x = 5) — дұрыс емес! == қолданыңыз
• if (x == true) — артық! if (x) жеткілікті
• String салыстыру үшін .equals() методын қолданыңыз
• Жақшаларсыз жазу қауіпті: if (condition) statement1; statement2;

🔶 switch Операторы

switch операторы — бір айнымалының көптеген мәндерін тексеруге арналған

Жалпы синтаксис:

switch (айнымалы) {
    case мән1:
        // Код
        break;
    case мән2:
        // Код
        break;
    default:
        // Әдепкі код
        break;
}

🔶 switch Операторының Мысалдары

📅 Апта күндері мысалы
// Апта күнін анықтау int dayNumber = 3; String dayName; switch (dayNumber) { case 1: dayName = "Дүйсенбі"; break; case 2: dayName = "Сейсенбі"; break; case 3: dayName = "Сәрсенбі"; break; case 4: dayName = "Бейсенбі"; break; case 5: dayName = "Жұма"; break; case 6: dayName = "Сенбі"; break; case 7: dayName = "Жексенбі"; break; default: dayName = "Дұрыс емес күн нөмірі"; break; } System.out.println("Бүгін: " + dayName);
🎓 Баға жүйесі мысалы
// Баға бойынша сипаттама char grade = 'B'; switch (grade) { case 'A': System.out.println("Өте жақсы! 90-100%"); break; case 'B': System.out.println("Жақсы! 80-89%"); break; case 'C': System.out.println("Қанағаттанарлық! 70-79%"); break; case 'D': System.out.println("Нашар! 60-69%"); break; case 'F': System.out.println("Тапсырмады! 0-59%"); break; default: System.out.println("Дұрыс емес баға"); break; }
🍂 Fall-through мысалы (break жоқ)
// Мезгіл анықтау (break жоқ мысал) int month = 3; String season; switch (month) { case 12: case 1: case 2: season = "Қыс"; break; case 3: case 4: case 5: season = "Көктем"; break; case 6: case 7: case 8: season = "Жаз"; break; case 9: case 10: case 11: season = "Күз"; break; default: season = "Дұрыс емес ай"; break; } System.out.println("Мезгіл: " + season);

⚠️ switch Операторының Ерекшеліктері:

• break міндетті! Болмаса келесі case-тер де орындалады
• Тек int, char, String, enum типтерін қолдауға болады
• default блогы міндетті емес, бірақ ұсынылады
• case мәндері тұрақты (constant) болуы керек

🔷 Үшлік Оператор (?:)

Үшлік оператор — қысқа if-else операторы

синтаксис: шарт ? мән1 : мән2

🔄 if-else дәстүрлі жолы

int age = 20;
String status;

if (age >= 18) {
    status = "Ересек";
} else {
    status = "Кәмелетке толмаған";
}

⚡ Үшлік операторы

int age = 20;

String status = age >= 18 ?
    "Ересек" :
    "Кәмелетке толмаған";

🔷 Үшлік Оператор Мысалдары

⚡ Үшлік операторының қолданылуы
// 1. Қарапайым мысал int a = 10, b = 20; int max = (a > b) ? a : b; System.out.println("Максимум: " + max); // 20 // 2. String-мен жұмыс int score = 85; String result = (score >= 60) ? "Өтті" : "Қанағаттанарлық"; System.out.println(result); // Өтті // 3. Методтарды шақыру boolean isRaining = true; System.out.println(isRaining ? "Қолшатыр алыңыз" : "Керемет ауа-райы"); // 4. Математикалық операциялар int number = -5; int absolute = (number < 0) ? -number : number; System.out.println("Абсолют мәні: " + absolute); // 5 // 5. Ұяландырылған үшлік операторлар int grade = 85; String letterGrade = (grade >= 90) ? "A" : (grade >= 80) ? "B" : (grade >= 70) ? "C" : "F"; System.out.println("Баға: " + letterGrade); // B

💡 Үшлік оператор ерекшеліктері:

• Қысқа if-else-тың баламасы
• Мән қайтарады (айнымалыға тағайындауға болады)
• Тек қарапайым шарттар үшін қолданыңыз
• Ұяландырылған түрі түсініксіз болуы мүмкін

🏗️ Ұяландырылған Таңдау Операторлары

// Ұяландырылған if операторлары
int temperature = 25;
boolean isSunny = true;

if (temperature > 0) {
    if (temperature > 30) {
        System.out.println("Ыстық күн");
        if (isSunny) {
            System.out.println("Серуенге жақсы");
        }
    } else {
        System.out.println("Жылы күн");
    }
} else {
    System.out.println("Суық күн");
}

⚠️ Ұяландырылған операторлар туралы кеңес:

• Терең ұяландыруды болдырмаңыз (3 деңгейден артық)
• Күрделі логиканы бөлек методтарға бөліңіз
• Жақшаларды дұрыс қолданыңыз
• Кодты оқуға жеңіл етіп жазыңыз

📊 Операторларды Салыстыру

Оператор Қолданылуы Артықшылығы Кемшілігі Мысал жағдай
if-else Универсальды Икемді, кез келген шарт Көп код жазу керек Күрделі логика
switch Көп таңдау Оқуға жеңіл, жылдам Тек белгілі типтерге Мәзір жүйесі
Үшлік ?: Қысқа шарт Компактты Түсініксіз болуы мүмкін Қарапайым таңдау

🎯 Практикалық Мысалдар

🏦 Банк жүйесі

double balance = 1500.0;
double amount = 200.0;

if (amount > balance) {
    System.out.println("Жеткіліксіз қаражат");
} else {
    balance -= amount;
    System.out.println("Операция сәтті");
}

🎮 Ойын жүйесі

int level = 15;
String difficulty;

switch (level / 5) {
    case 0: difficulty = "Жеңіл"; break;
    case 1: difficulty = "Орташа"; break;
    case 2: difficulty = "Қиын"; break;
    default: difficulty = "Эксперт";
}

🌡️ Ауа-райы жүйесі

int temp = 25;
String clothes = (temp > 20) ?
    (temp > 30 ? "Жұқа киім" : "Жеңіл киім") :
    (temp > 0 ? "Жылы киім" : "Қыстық киім");

System.out.println("Ұсыныс: " + clothes);

✅ Жақсы Тәжірибелер

✅ Дұрыс тәжірибелер

// 1. Жақшаларды әрқашан қолданыңыз
if (condition) {
    doSomething();
}

// 2. String салыстыруда equals()
if (name.equals("Айдос")) {
    // код
}

// 3. switch-те break қолданыңыз
switch (value) {
    case 1:
        action();
        break;
    default:
        defaultAction();
}

❌ Жаман тәжірибелер

// 1. Жақшасыз (қауіпті!)
if (condition)
    doSomething();
    doSomethingElse(); // әрқашан орындалады!

// 2. = операторын == орнына
if (x = 5) { // ҚАТЕ!
    // код
}

// 3. break жоқ
switch (value) {
    case 1:
        action1(); // келесіге де өтеді!
    case 2:
        action2();
}

💡 Маңызды Кеңестер

🎯 Дұрыс оператор таңдаңыз

if — күрделі шарттарға, switch — көп мәнді салыстыруға, үшлік оператор — қысқа таңдауға қолданыңыз.

🔒 Жақшаларды қолданыңыз

Тіпті бір жол код болса да, жақшаларды {} қолданыңыз. Бұл қателерді болдырмайды.

📝 String салыстыру

String айнымалыларын салыстыру үшін == емес, .equals() методын қолданыңыз.

🔄 break операторы

switch операторында break қолдануды ұмытпаңыз, әйтпесе келесі case-тер де орындалады.

🧠 Логикалық операторлар

Күрделі шарттарда &&, ||, ! операторларын дұрыс қолданыңыз және жақшалармен топтаңыз.

📊 Кодтың оқылуы

Кодыңыз түсінікті болуы керек. Терең ұяландыруды болдырмаңыз, код құрылымын жақсартыңыз.
