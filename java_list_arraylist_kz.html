List — өлшемі өзгеретін элементтер жиымы

🔍 List және ArrayList туралы негізгі түсініктер

List - бұл элементтердің реттелген жиымын сақтайтын интерфейс

📋 List Interface

Элементтерді реттелген түрде сақтайтын интерфейс

Абстрактілі түсінік

🔧 ArrayList Class

List интерфейсінің динамикалық массив негіздегі іске асуы

Практикалық іске асыру

🎯 Generic Type

List<String>, List<Integer> сияқты типтерді көрсету

Типтік қауіпсіздік

⚖️ Array vs ArrayList салыстыруы

🔒 Array (Массив)

  • Тұрақты өлшем
  • Жылдам қол жеткізу
  • Аз жады шығыны
  • Примитивті типтерді қолдайды
  • Өлшемін өзгерту мүмкін емес
int[] arr = new int[5];

🔄 ArrayList

  • Динамикалық өлшем
  • Автоматты кеңейту
  • Көптеген пайдалы методтар
  • Тек объекттерді сақтайды
  • Элементтерді қосу/жою оңай
ArrayList<Integer> list = new ArrayList<>();

🚀 ArrayList құру және инициализациялау

📝 ArrayList құру әдістері
// import жасау import java.util.ArrayList; import java.util.List; // 1. Бос ArrayList құру ArrayList<String> names = new ArrayList<>(); // 2. List интерфейсі арқылы List<Integer> numbers = new ArrayList<>(); // 3. Бастапқы сыйымдылықпен ArrayList<String> cities = new ArrayList<>(100); // 4. Бастапқы мәндермен (Java 9+) List<String> colors = List.of("қызыл", "жасыл", "көк"); // 5. Arrays.asList() арқылы List<String> fruits = Arrays.asList("алма", "банан", "апельсин");

💡 Жақсы практика:

• List интерфейсін пайдаланыңыз: List<String> list = new ArrayList<>();
• Diamond operator (<>) пайдаланыңыз Java 7+ версияларында
• Егер элементтер саны белгілі болса, бастапқы сыйымдылықты көрсетіңіз

🛠️ ArrayList негізгі методтары

➕ Элемент қосу

add(element)
add(index, element)

Жиымға элемент қосады

🔍 Элемент алу

get(index)

Көрсетілген индекстегі элементті қайтарады

✏️ Элемент өзгерту

set(index, element)

Көрсетілген индекстегі элементті өзгертеді

❌ Элемент жою

remove(index)
remove(element)

Элементті жояды

📏 Өлшемді білу

size()

Жиымдағы элементтер санын қайтарады

❓ Бос екенін тексеру

isEmpty()

Жиым бос болса true қайтарады

🔎 Элемент іздеу

contains(element)
indexOf(element)

Элементтің бар/жоқ екенін тексереді

🧹 Тазалау

clear()

Барлық элементтерді жояды

💻 ArrayList методтарының мысалдары

🚀 Толық мысал - Студенттер тізімі
import java.util.ArrayList; import java.util.List; public class StudentListExample { public static void main(String[] args) { // Студенттер тізімін құру List<String> students = new ArrayList<>(); // Студенттерді қосу students.add("Айгерім"); students.add("Арман"); students.add("Дана"); students.add("Ерлан"); // Студенттер санын көрсету System.out.println("Студенттер саны: " + students.size()); // Бірінші студентті алу String firstStudent = students.get(0); System.out.println("Бірінші студент: " + firstStudent); // Студентті ауыстыру students.set(1, "Асхат"); // Студентті іздеу boolean hasDana = students.contains("Дана"); System.out.println("Дана бар ма? " + hasDana); // Студентті жою students.remove("Дана"); students.remove(0); // Индекс бойынша жою // Қалған студенттерді көрсету System.out.println("Қалған студенттер: " + students); } }

🎯 Generic типтермен жұмыс

Generic-тер типтік қауіпсіздікті қамтамасыз етеді

📝 String тізімі

List<String> names

Тек String мәндерін сақтайды

🔢 Integer тізімі

List<Integer> numbers

Тек Integer мәндерін сақтайды

👤 Object тізімі

List<Person> people

Тек Person объекттерін сақтайды

❓ Raw Type (ұсынылмайды)

List list

Кез келген типті сақтайды

⚠️ Raw Type пайдаланбаңыз!

Generic көрсетпеген List пайдалану компиляция кезінде қате тудырмайды, бірақ runtime кезінде ClassCastException қатесін тудыруы мүмкін.
