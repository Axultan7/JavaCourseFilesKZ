🔢 Java Бүтін Сандар Типтері

byte, short, int, long - толық нұсқаулық

Бүтін сан - ондық бөлігі жоқ сан (1, 25, -100, 0)

📊 Бүтін Сандар Типтерінің Шолуы

Java-да 4 түрлі бүтін сан типі бар. Олардың арасындағы айырмашылық - сақтай алатын сан диапазоны мен жад мөлшері.

byte

Мөлшері: 1 байт (8 бит)
Диапазон:
-128 ... 127
Қолданылуы:
Кішкентай сандар үшін

short

Мөлшері: 2 байт (16 бит)
Диапазон:
-32,768 ... 32,767
Қолданылуы:
Орташа сандар үшін

int

Мөлшері: 4 байт (32 бит)
Диапазон:
-2,147,483,648 ... 2,147,483,647
Қолданылуы:
Ең көп қолданылатын тип

long

Мөлшері: 8 байт (64 бит)
Диапазон:
-9,223,372,036,854,775,808 ...
9,223,372,036,854,775,807
Қолданылуы:
Өте үлкен сандар үшін

Тип Мөлшері Минимум мәні Максимум мәні Мысал Ерекшеліктері

💻 Айнымалыларды Жариялау

📝 Бүтін сандар типтерінің мысалдары
// byte - кішкентай сандар үшін byte studentAge = 20; byte temperature = -5; byte grade = 100; // short - орташа сандар үшін short currentYear = 2024; short elevation = 1500; short populationInThousands = 25000; // int - стандартты тип int populationCount = 20000000; int bankAccount = 1500000; int secondsInDay = 86400; // long - үлкен сандар үшін (L жұрнағымен) long earthToSunDistance = 149597870700L; long phoneNumber = 87051234567L; long fileSize = 5368709120L;

💡 Маңызды ережелер:

• long типіне мән беру кезінде соңына L немесе l жұрнағын қосыңыз
• Әдетте int типін қолданыңыз, егер аса үлкен сан қажет болмаса
• byte және short жад үнемдеу үшін қолданылады
• Бүтін сандарда ондық нүкте болмайды

⚠️ Overflow (Толып кету) проблемасы

Айнымалыға оның сақтай алатын диапазонынан үлкен мән берсе не болады?

Мысал:
byte maxValue = 127; // byte-тың максимумы
maxValue++; // 128 болуы керек еді, бірақ...
// Нәтиже: -128 (минимумға секіреді!)
int типі үшін:
int maxInt = 2147483647; // int максимумы
maxInt = maxInt + 1; // 2147483648 болуы керек еді
// Нәтиже: -2147483648 (минимумға секіреді!)
