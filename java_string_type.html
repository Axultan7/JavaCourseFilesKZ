📝 Java String Типі

Мәтінмен жұмыс істеудің толық нұсқаулығы

String - таңбалар тізбегі ("Сәлем, Әлем!")

📖 String дегеніміз не?

String — бұл Java-дағы ең маңызды класстардың бірі. Мәтінмен жұмыс істеу үшін қолданылады.

🔗 Immutable

Өзгермейтін — бір рет құрылғаннан кейін өзгерту мүмкін емес

📚 Reference Type

Сілтемелі тип — объект ретінде сақталады

🌍 Unicode

Кез келген тілдің таңбаларын қолдайды

⚡ Optimized

String Pool арқылы оптималданған

💻 String Айнымалыларын Жариялау

📝 String құру әдістері
// 1. Литерал арқылы (ең жиі қолданылатын) String name = "Айдос"; String city = "Алматы"; String greeting = "Сәлем!"; // 2. new операторы арқылы String country = new String("Қазақстан"); String university = new String("ҚазҰТЗУ"); // 3. Бос жол құру String empty1 = ""; // Бос литерал String empty2 = new String(); // Бос конструктор // 4. char массивінен құру char[] letters = {'J', 'a', 'v', 'a'}; String fromArray = new String(letters); // "Java" // 5. StringBuilder арқылы StringBuilder sb = new StringBuilder("Hello"); String fromBuilder = sb.toString();

⚠️ Литерал vs new операторы:

• String s = "Hello"; — String Pool-да сақталады, жадыны үнемдейді
• String s = new String("Hello"); — жаңа объект құрады, көп жад алады
• Әдетте литерал әдісін қолданыңыз

🔒 String Immutability (Өзгермейтіндігі)

String объектілері өзгермейді. Өзгерту операциялары жаңа String құрады.

String str = "Java";
str = str + " Programming"; // Жаңа String құрылады
// Ескі "Java" объектісі жойылады

Алғашқы

str → "Java"

Конкатенациядан кейін

str → "Java Programming"

"Java" жойылды

🛠️ String Методтары

length()

Жолдың ұзындығын қайтарады

"Сәлем".length() → 5
charAt(index)

Белгілі позициядағы таңбаны қайтарады

"Java".charAt(0) → 'J'
substring(start, end)

Жолдың бөлігін алады

"Programming".substring(0, 4) → "Prog"
toLowerCase()

Кіші әріптерге өзгертеді

"JAVA".toLowerCase() → "java"
toUpperCase()

Бас әріптерге өзгертеді

"java".toUpperCase() → "JAVA"
trim()

Басы мен соңындағы бос орындарды жояды

" Java ".trim() → "Java"
replace(old, new)

Белгілі бөлікті ауыстырады

"Java".replace("a", "o") → "Jovo"
contains(str)

Жолда басқа жол бар ма тексереді

"Java Programming".contains("Program") → true
startsWith(prefix)

Белгілі префикспен басталады ма тексереді

"Java".startsWith("Ja") → true
endsWith(suffix)

Белгілі суффикспен аяқталады ма тексереді

"Java".endsWith("va") → true
split(delimiter)

Жолды бөліктерге бөледі

"a,b,c".split(",") → ["a", "b", "c"]
isEmpty()

Жол бос па тексереді

"".isEmpty() → true

🔗 String Конкатенациясы

➕ Жолдарды біріктіру әдістері
// 1. + операторы арқылы String firstName = "Айдос"; String lastName = "Серікұлы"; String fullName = firstName + " " + lastName; // "Айдос Серікұлы" // 2. += операторы арқылы String message = "Сәлем"; message += " Әлем!"; // "Сәлем Әлем!" // 3. concat() методы арқылы String greeting = "Қош".concat(" келдіңіз!"); // "Қош келдіңіз!" // 4. Сандармен қосу int age = 25; String info = "Жасы: " + age; // "Жасы: 25" // 5. String.format() арқылы String formatted = String.format("Аты: %s, Жасы: %d", "Айгүл", 22); // "Аты: Айгүл, Жасы: 22"

💡 Конкатенация ережелері:

• String + кез келген тип = String
• Солдан оңға қарай орындалады
• null + String = "nullString"
• Көп конкатенация үшін StringBuilder қолданыңыз
