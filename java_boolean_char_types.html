✅ Java Логикалық және Символдық Типтері

boolean және char - ақиқат пен таңбалар

boolean: true/false    char: бір таңба

🔍 Типтердің Шолуы

boolean

Мөлшері: 1 бит (теория бойынша)
Мәндер: тек true немесе false
Қолданылуы: логикалық шешімдер
Мысал: boolean isReady = true;

char

Мөлшері: 2 байт (16 бит)
Кодталуы: Unicode (UTF-16)
Диапазон: 0 ... 65,535
Мысал: char letter = 'А';

✅ boolean Типі

boolean - ең қарапайым мәліметтер типі. Тек екі мән ала алады: true (ақиқат) немесе false (жалған).

true

Ақиқат

Логикалық 1

false

Жалған

Логикалық 0

🎯 Қолданылуы:

• Шартты операторларда (if, while)
• Флагтар ретінде (isActive, hasPermission)
• Логикалық операциялар нәтижесі ретінде

💻 boolean Айнымалыларын Жариялау

📝 boolean типінің мысалдары
// Тікелей мән беру boolean isStudent = true; boolean isMarried = false; boolean hasLicense = true; // Салыстыру операциялары арқылы int age = 25; boolean isAdult = age >= 18; // true boolean isTeenager = age < 20; // false // Логикалық операциялар boolean canVote = isAdult && hasLicense; // true && true = true boolean needsHelp = !isAdult || isStudent; // !true || true = true // Әдістер арқылы String text = "Java"; boolean isEmpty = text.isEmpty(); // false boolean startsWith = text.startsWith("Ja"); // true

💡 Маңызды ескертулер:

• Java-да 0 және 1 boolean мәні емес (C/C++ сияқты емес)
• true/false кілт сөздері - тек осылар ғана boolean мәндері
• Boolean объектісі де бар, бірақ boolean примитиві жылдамырақ
• Шартты операторларда тек boolean типі қолданылады

🔗 Логикалық Операторлар

🧮 Логикалық операциялар
boolean a = true; boolean b = false; // AND (ЖӘНЕ) - && boolean result1 = a && b; // false (екеуі де true болуы керек) // OR (НЕМЕСЕ) - || boolean result2 = a || b; // true (біреуі true болса жеткілікті) // NOT (ЕМЕС) - ! boolean result3 = !a; // false (керісінше мән) // XOR (АЙЫРЫМ) - ^ boolean result4 = a ^ b; // true (әртүрлі болса true)

🔤 char Типі

char - бір таңбаны сақтайтын тип. Unicode стандартын қолданады, кез келген тілдің таңбаларын сақтай алады.

Латын әріптері

'A' 'z' 'X'

char a = 'A';

Қазақ әріптері

'А' 'ә' 'ұ'

char kaz = 'ә';

Цифрлар

'0' '5' '9'

char digit = '5';

Арнайы таңбалар

'@' '#' '&'

char symbol = '@';

Эмодзилар

'😊' '🚀' '❤'

char emoji = '😊';

Басқа тілдер

'中' 'أ' 'λ'

char foreign = '中';

💻 char Айнымалыларын Жариялау

📝 char типінің мысалдары
// Жай таңбалар char letter = 'A'; char kazakh = 'ә'; char digit = '5'; char symbol = '@'; // Unicode кодтары арқылы char unicodeA = 65; // 'A' таңбасы char unicodeHeart = 9829; // '♥' таңбасы // Unicode escape sequence арқылы char escapeA = '\u0041'; // 'A' таңбасы char escapeCopyright = '\u00A9'; // '©' таңбасы // Арнайы таңбалар (escape sequences) char newline = '\n'; // Жаңа жол char tab = '\t'; // Табуляция char backslash = '\\'; // Кері сызық char quote = '\''; // Бір тырнақша
