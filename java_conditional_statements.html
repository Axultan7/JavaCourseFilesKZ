if, else if, else — шешім қабылдау операторлары

Шартты оператор — шартқа байланысты код орындайды

🤔 Шартты Операторлар дегеніміз не?

Шартты операторлар бағдарламаға шешім қабылдау мүмкіндігін береді. Шартқа байланысты әртүрлі код блоктары орындалады.

🔵 if

Негізгі шарт

Шарт ақиқат болса орындалады

🔶 else if

Қосымша шарттар

Алғашқы шарт жалған болса тексереді

🔷 else

Әдепкі нұсқа

Ешбір шарт орындалмаса іске қосылады

🔵 Қарапайым if Операторы

Синтаксис:

if (шарт) {
    // Шарт ақиқат болса орындалатын код
}

💻 Қарапайым if Мысалдары

📝 Негізгі if операторы мысалдары
// 1. Сан тексеру int age = 20; if (age >= 18) { System.out.println("Сіз ересек адамсыз"); } // 2. Логикалық айнымалымен boolean isRaining = true; if (isRaining) { System.out.println("Күнә алып шығыңыз"); } // 3. String тексеру String password = "123456"; if (password.length() >= 6) { System.out.println("Пароль жеткілікті ұзын"); } // 4. Арифметикалық тексеру double balance = 1500.50; double price = 999.99; if (balance >= price) { System.out.println("Сатып алу мүмкін"); } // 5. Методтың нәтижесін тексеру String name = "Айдос"; if (name.startsWith("А")) { System.out.println("Аты А әрпімен басталады"); }

💡 if Операторының Ерекшеліктері:

• Шарт міндетті түрде boolean мәнін қайтаруы керек
• Жақшалар {} қолдану міндетті емес, бірақ ұсынылады
• Шарт жалған болса, if блогы толығымен өткізіледі
• Бір жолдық код болса да жақшаларды қолданған жөн

🔶 if-else Операторы

Синтаксис:

if (шарт) {
    // Шарт ақиқат болса орындалатын код
} else {
    // Шарт жалған болса орындалатын код
}

🎯 if-else-тің артықшылығы:

else блогы бар болғандықтан, міндетті түрде бір код блогы орындалады. Бұл кодтың болжамды болуын қамтамасыз етеді.

🔄 if-else Мысалдары

📝 if-else операторының мысалдары
// 1. Жас тексеру int age = 16; if (age >= 18) { System.out.println("Сіз ересек адамсыз"); } else { System.out.println("Сіз кәмелетке толмағансыз"); } // 2. Жұп/тақ сан анықтау int number = 17; if (number % 2 == 0) { System.out.println(number + " - жұп сан"); } else { System.out.println(number + " - тақ сан"); } // 3. Максимум табу int a = 25, b = 30; int max; if (a > b) { max = a; } else { max = b; } System.out.println("Максимум: " + max); // 4. Ауа-райы бойынша кеңес int temperature = 5; if (temperature > 15) { System.out.println("Жеңіл киім киіңіз"); } else { System.out.println("Жылы киім киіңіз"); } // 5. Пароль тексеру String inputPassword = "mypassword"; String correctPassword = "secretword"; if (inputPassword.equals(correctPassword)) { System.out.println("Кіру рұқсат етілді"); } else { System.out.println("Қате пароль"); }

✅ if-else пайдалану кеңестері:

• Әрқашан екі нұсқа болса if-else қолданыңыз
• Boolean айнымалыларды тікелей қолданыңыз: if (flag) емес if (flag == true)
• String салыстыру үшін .equals() методын қолданыңыз
• Кодтың оқылуын жақсарту үшін жақшаларды қолданыңыз

🔷 else if — Көп Шартты Тексеру

Синтаксис:

if (шарт1) {
    // Шарт1 ақиқат болса
} else if (шарт2) {
    // Шарт2 ақиқат болса
} else if (шарт3) {
    // Шарт3 ақиқат болса
} else {
    // Ешбір шарт орындалмаса
}

🔍 else if жұмысының принципі:

• Шарттар жоғарыдан төменге қарай тексеріледі
• Алғашқы ақиқат шарт табылған кезде, қалғандары тексерілмейді
• Тек бір блок орындалады
• else блогы міндетті емес
