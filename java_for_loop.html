💻 Негізгі for Цикл Мысалдары

📝 Қарапайым for цикл мысалдары
// 1. 1-ден 10-ға дейін сандарды шығару for (int i = 1; i <= 10; i++) { System.out.println("Сан: " + i); } // 2. Кері санау (10-нан 1-ге дейін) for (int i = 10; i >= 1; i--) { System.out.println("Қалды: " + i); } // 3. Жұп сандарды шығару for (int i = 0; i <= 20; i += 2) { System.out.println("Жұп сан: " + i); } // 4. Сандардың қосындысын табу int sum = 0; for (int i = 1; i <= 100; i++) { sum += i; } System.out.println("1-ден 100-ге дейінгі сандардың қосындысы: " + sum); // 5. Факториал есептеу int n = 5; int factorial = 1; for (int i = 1; i <= n; i++) { factorial *= i; } System.out.println(n + "! = " + factorial); // 5! = 120

💡 for Циклының Артықшылықтары:

• Есептегіш айнымалысы автоматты түрде басқарылады
• Қайталау саны алдын ала белгілі болса өте ыңғайлы
• Компактты және оқуға жеңіл синтаксис
• Массивтер мен коллекциялармен жұмыс істеуге өте қолайлы

Шексіз цикл (абайлаңыз!) /* for (;;) { System.out.println("Шексіз цикл!"); // break; - циклден шығу үшін } */ // 5. Бос бөлімдермен int count = 0; for (; count < 5; ) { System.out.println("Count: " + count); count++; // Қолмен арттыру }

⚠️ for Циклында Жиі Кездесетін Қателер:

• Шексіз цикл: шарт ешқашан false болмайды
• Off-by-one қатесі: < пен <= арасындағы айырма
• Есептегішті өзгерту: цикл ішінде i мәнін қолмен өзгерту
• Нүктелі үтір: for(;;) дұрыс, for();//— қате

🚀 Enhanced for Цикл (for-each)

Enhanced for синтаксисі:

for (тип айнымалы : коллекция) {
    // Әр элементпен жұмыс
}

✨ Enhanced for-тің артықшылықтары:

• Қарапайым және оқуға жеңіл
• Индекспен қате жіберу мүмкіндігі жоқ
• Автоматты түрде барлық элементтерді өтеді
• Массивтер мен коллекциялармен керемет жұмыс істейді
