📝 Негізгі арифметикалық операциялар
// Қарапайым арифметика int a = 10; int b = 3; int sum = a + b; // 13 int difference = a - b; // 7 int product = a * b; // 30 int quotient = a / b; // 3 (бүтін бөлу) int remainder = a % b; // 1 (қалдық) // Нақты сандармен double x = 10.0; double y = 3.0; double result = x / y; // 3.3333333333333335 // Аралас типтермен int intVal = 7; double doubleVal = 2.5; double mixed = intVal + doubleVal; // 9.5

⚠️ Бүтін сандарды бөлу:

• int / int = int (ондық бөлігі жойылады)
• 7 / 2 = 3 (3.5 емес!)
• Ондық нәтиже алу үшін кемінде бір мәнді double етіп жазыңыз
• 7.0 / 2 = 3.5 немесе 7 / 2.0 = 3.5

📊 Салыстыру Операторлары

==

Тең

5 == 5 → true

!=

Тең емес

5 != 3 → true

<

Кіші

3 < 5 → true

>

Үлкен

5 > 3 → true

<=

Кіші немесе тең

3 <= 5 → true

>=

Үлкен немесе тең

5 >= 5 → true

int age = 25;
boolean isAdult = age >= 18; // true
boolean isTeenager = age < 20; // false
boolean isExact = age == 25; // true
