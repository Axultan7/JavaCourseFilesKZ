🔄 while және do-while дегеніміз не?

while циклдары шарт ақиқат болған кезде код блогын қайталайды. Қайталау саны алдын ала белгісіз болғанда қолданылады.

Әртүрлі жағдайларға арналған

for — белгілі сан, while — белгісіз

🔄 while Циклы

while синтаксисі:

while (шарт) {
    // Қайталанатын код
    // Шартты өзгертетін код
}

🔍 while Циклының Ерекшеліктері:

• Алдымен шартты тексереді
• Шарт жалған болса, код блогы мүлдем орындалмауы мүмкін
• Цикл ішінде шартты өзгертетін код болуы керек
• Шексіз цикл қаупі бар

📝 while циклының негізгі мысалдары
// 1. 1-ден 10-ға дейін санау int i = 1; while (i <= 10) { System.out.println("Сан: " + i); i++; // Міндетті! Әйтпесе шексіз цикл } // 2. Пайдаланушы енгізуін күту Scanner scanner = new Scanner(System.in); String input = ""; while (!input.equals("шығу")) { System.out.print("Команда енгізіңіз (шығу үшін 'шығу'): "); input = scanner.nextLine(); if (!input.equals("шығу")) { System.out.println("Сіз енгіздіңіз: " + input); } } // 3. Санды табу ойыны int secretNumber = 7; int guess = -1; int attempts = 0; while (guess != secretNumber) { System.out.print("Санды болжаңыз (1-10): "); guess = scanner.nextInt(); attempts++; if (guess < secretNumber) { System.out.println("Үлкенірек ойлаңыз!"); } else if (guess > secretNumber) { System.out.println("Кішірек ойлаңыз!"); } else { System.out.println("Дұрыс! " + attempts + " әрекетте таптыңыз!"); } } // 4. Факториал есептеу int n = 5; int factorial = 1; int counter = 1; while (counter <= n) { factorial *= counter; counter++; } System.out.println(n + "! = " + factorial); // 5. Цифрлардың қосындысын табу int number = 12345; int sum = 0; while (number > 0) { sum += number % 10; // Соңғы цифр number /= 10; // Соңғы цифрды жою } System.out.println("Цифрлардың қосындысы: " + sum); // 15

💡 while Циклын Қашан Қолдану Керек:

• Қайталау саны алдын ала белгісіз болғанда
• Пайдаланушы енгізуін күтуде
• Белгілі бір шартқа жеткенше жұмыс істеуде
• Файлдан оқуда (соңына жеткенше)
• Ойындарда (ойын аяқталғанша)

🔁 do-while Циклы

do-while синтаксисі:

do {
    // Қайталанатын код
    // Шартты өзгертетін код
} while (шарт);

Мәзір жүйесі Scanner scanner = new Scanner(System.in); int choice; do { System.out.println("\n=== НЕГІЗГІ МӘЗІР ==="); System.out.println("1. Файл ашу"); System.out.println("2. Файл сақтау"); System.out.println("3. Баптау"); System.out.println("0. Шығу"); System.out.print("Таңдауыңыз: "); choice = scanner.nextInt(); switch (choice) { case 1: System.out.println("Файл ашылуда..."); break; case 2: System.out.println("Файл сақталуда..."); break; case 3: System.out.println("Баптау ашылуда..."); break; case 0: System.out.println("Бағдарлама аяқталуда..."); break; default: System.out.println("Дұрыс емес таңдау!"); } } while (choice != 0);
