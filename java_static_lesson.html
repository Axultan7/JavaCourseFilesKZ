⚡ Java Static Кілт Сөзі

Класс деңгейіндегі айнымалылар мен әдістерді үйрену

Static: объектсіз қолдану    Class-level: класс деңгейі

🎯 Static дегеніміз не?

Static кілт сөзі айнымалылар мен әдістерді класс деңгейінде жасау үшін қолданылады. Static мүшелер объект жасамай-ақ қолданыла алады.

🧠 Жадыдағы Static vs Instance

Static мүшелер класс жүктелгенде бір рет жасалады, ал instance мүшелер әр объект үшін жеке жасалады.

📦 Static Memory

• Method Area-да сақталады
• Класс жүктелгенде жасалады
• Барлық объектілер үшін ортақ
• Программа аяқталғанша өмір сүреді

🏠 Instance Memory

• Heap-те сақталады
• Объект жасалғанда жасалады
• Әр объект үшін жеке
• Garbage Collector тазалайды

🔗 Қатынау жолдары

• Static: ClassName.memberName
• Instance: objectName.memberName
• Static-тен instance-ға қатынау жоқ
• Instance-тан static-қа қатынау бар

⏰ Өмір сүру уақыты

• Static: программа басынан аяғына
• Instance: объект жасалғаннан жойылғанға
• Static алдымен жүктеледі
• Instance кейінірек жасалады

📊 Static Variables (Класс Айнымалылары)

📈 Static variable мысалы - Студент санағышы
public class Student { // Static айнымалы - барлық студенттер үшін ортақ private static int totalStudents = 0; private static String universityName = "ҚазҰУ"; // Instance айнымалылар - әр объект үшін жеке private String name; private int id; // Конструктор public Student(String name, int id) { this.name = name; this.id = id; totalStudents++; // Студент саны артады System.out.println("Жаңа студент қосылды. Жалпы саны: " + totalStudents); } // Static әдіс - объектсіз шақыруға болады public static int getTotalStudents() { return totalStudents; } // Static әдіс - университет атауын алу public static String getUniversityName() { return universityName; } // Static әдіс - университет атауын өзгерту public static void setUniversityName(String newName) { universityName = newName; System.out.println("Университет аты өзгертілді: " + newName); } // Instance әдіс public void displayInfo() { System.out.println("ID: " + id + ", Аты: " + name + ", Университет: " + universityName); } }

💡 Маңызды ақпарат!

Static айнымалылар барлық объектілер арасында бөлісіледі. Бір объектте өзгерту басқа барлық объектілерге әсер етеді!

🔧 Static Methods (Класс Әдістері)

Static әдістер объект жасамай-ақ шақырыла алады. Олар тек static мүшелерге ғана қатынай алады.

🧮 Utility класс мысалы

public class MathUtils { // Static константалар public static final double PI = 3.14159; public static final int MAX_VALUE = 1000; // Static әдістер public static int max(int a, int b) { return (a > b) ? a : b; } public static double circleArea(double radius) { return PI * radius * radius; } public static boolean isEven(int number) { return number % 2 == 0; } } // Қолдану int result = MathUtils.max(10, 20); double area = MathUtils.circleArea(5.0);

🏦 Counter класс мысалы

public class Counter { private static int count = 0; // Static әдістер public static void increment() { count++; System.out.println("Count: " + count); } public static void decrement() { count--; System.out.println("Count: " + count); } public static int getCount() { return count; } public static void reset() { count = 0; System.out.println("Counter reset edildi"); } } // Қолдану Counter.increment(); // 1 Counter.increment(); // 2 Counter.decrement(); // 1

⚠️ Static әдістердің шектеулері

• this және super кілт сөздерін қолдана алмайды
• Instance айнымалыларға тікелей қатынай алмайды
• Instance әдістерді тікелей шақыра алмайды
• Overriding жасала алмайды (polymorphism жоқ)

public class Database { private static String connectionString; private static String username; private static String password; private static boolean isConnected = false; // Static блок 1 - конфигурация жүктеу static { System.out.println("1️⃣ Database конфигурациясын жүктеу..."); connectionString = "jdbc:mysql://localhost:3306/mydb"; username = "admin"; password = "password123"; System.out.println("✅ Конфигурация жүктелді"); } // Static блок 2 - байланыс орнату static { System.out.println("2️⃣ Database-қа қосылу..."); try { // Байланыс орнату симуляциясы Thread.sleep(1000); // 1 секунд күту isConnected = true; System.out.println("✅ Database қосылды"); } catch (Exception e) { System.out.println("❌ Қосылу қатесі: " + e.getMessage()); } } // Static әдіс public static void getConnectionInfo() { System.out.println("Connection: " + connectionString); System.out.println("Status: " + (isConnected ? "Қосылған" : "Қосылмаған")); } }

🎯 Static-тің Қолданылу Жерлері

Static элементтер қашан және қалай қолданылатынын үйренейік

🧮 Utility Classes

Math, Arrays, Collections сияқты көмекші кластар

Math.max(), Arrays.sort()

🏭 Factory Methods

Объект жасау үшін статикалық әдістер

String.valueOf(), Integer.parseInt()

📊 Constants

Тұрақты мәндер үшін

Math.PI, Integer.MAX_VALUE

📈 Counters

Барлық объектілер үшін ортақ санағыштар

totalObjects, instanceCount

🔧 Configuration

Қосымша параметрлері

appName, version, debugMode

🎯 Singleton Pattern

Бір ғана инстанс болуы керек кластар

getInstance() method

🏪 Singleton Pattern мысалы

public class Settings { // Жалғыз инстанс private static Settings instance; // Конфигурация мәліметтері private String theme; private String language; // Private конструктор private Settings() { theme = "Dark"; language = "Қазақша"; } // Static factory method public static Settings getInstance() { if (instance == null) { instance = new Settings(); } return instance; } // Getter/Setter әдістер public String getTheme() { return theme; } public void setTheme(String theme) { this.theme = theme; } }

🏦 BankAccount with Statistics

public class BankAccount { // Static статистика private static int totalAccounts = 0; private static double totalBalance = 0.0; private static final double INTEREST_RATE = 0.05; // Instance мәліметтер private String accountNumber; private double balance; public BankAccount(String accountNumber, double initialBalance) { this.accountNumber = accountNumber; this.balance = initialBalance; totalAccounts++; totalBalance += initialBalance; } // Static әдістер public static void printStatistics() { System.out.println("Жалпы шоттар саны: " + totalAccounts); System.out.println("Жалпы баланс: " + totalBalance); System.out.println("Орташа баланс: " + (totalAccounts > 0 ? totalBalance/totalAccounts : 0)); } }

⚠️ Static қолданудағы қателер

• Static блокта exception болса, ClassNotFound қатесі шығады
• Static әдістен instance мүшелерге тікелей қатынау
• Көп static айнымалы қолдану (жады проблемалары)
• Static import-ты дұрыс қолданбау
• Static nested class пен inner class арасындағы айырмашылықты білмеу

💡 Static Import туралы

Java 5-тен бастап static import қолдануға болады:
import static java.lang.Math.*;
Бұдан кейін Math.sqrt() орнына тек sqrt() жазуға болады.

🎯 Static қолданудағы ең жақсы тәжірибелер

🔧 Static мақсатын анықтау

• Utility функциялар үшін қолданыңыз
• Объектке тәуелді емес мәліметтер үшін
• Тұрақты мәндер үшін final екі қосыңыз

⚡ Performance ескеру

• Static блоктар класс жүктелгенде орындалады
• Көп static айнымалы жады қолданады
• Lazy initialization қолданыңыз

🔒 Thread Safety

• Static айнымалылар бірнеше thread арасында бөлісіледі
• Synchronization керек болуы мүмкін
• Immutable объектілерді қолданыңыз

🧪 Testing ескеру

• Static методтар test жасауды қиындатады
• Mock жасау қиын
• Dependency injection қолданыңыз
