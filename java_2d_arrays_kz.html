2D Массив — массивтардың массиві (матрица)

📋 2D Массив дегеніміз не?

Екіөлшемді массив — бұл жол мен бағанның қиылысуында орналасқан элементтердің жиынтығы. Ол кестелер, матрицалар, ойындардағы карталар сияқты деректерді сақтауға ыңғайлы.

🔍 2D Массивтің ерекшеліктері:

• Екі индекс арқылы қол жеткізу: [жол][баған]
• Бірінші индекс — жол нөмірі (row)
• Екінші индекс — баған нөмірі (column)
• Тік бұрышты немесе jagged болуы мүмкін
• Негізінде массивтардың массиві

📝 2D Массив Жариялау

Жалпы синтаксис:

тип[][] массивАты; // немесе
тип массивАты[][]; // немесе
тип[] массивАты[];

1️⃣ Жариялау

int[][] matrix;
String[][] table;
double[][] prices;

2️⃣ Мөлшер көрсету

matrix = new int[3][4];
table = new String[2][5];
prices = new double[3][3];

3️⃣ Бір жолда

int[][] matrix = new int[3][4];
String[][] table = new String[2][5];
double[][] prices = new double[3][3];

💡 Тұсінік:

• int[3][4] — 3 жол, 4 баған
• matrix.length — жолдар саны
• matrix[0].length — бірінші жолдағы бағандар саны
• Жолдардың ұзындығы әртүрлі болуы мүмкін (jagged arrays)

🎯 2D Массивті Инициализациялау

🔹 Мәндермен инициализация

int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

🔹 Бос массив құру

int[][] matrix = new int[3][3];
// Барлық элементтер 0-ге тең

String[][] names = new String[2][3];
// Барлық элементтер null

🔹 Кейінгі инициализация

int[][] matrix = new int[2][3];
matrix[0][0] = 10;
matrix[0][1] = 20;
matrix[1][2] = 30;

🔹 Бір жолда толық инициализация

String[][] schedule = {

💻 2D Массив Негіздері Мысалдары

📝 2D массив жариялау және қолдану
// 1. 2D массив жариялау және инициализациялау int[][] matrix = { {1, 2, 3}, {4, 5, 6}, {7, 8, 9} }; String[][] subjects = { {"Математика", "Физика"}, {"Тарих", "География"}, {"Қазақ тілі", "Ағылшын"} }; // 2. Өлшемдерді алу System.out.println("Жолдар саны: " + matrix.length); // 3 System.out.println("Бағандар саны: " + matrix[0].length); // 3 // 3. Элементтерге қол жеткізу System.out.println("Бірінші элемент: " + matrix[0][0]); // 1 System.out.println("Орталық элемент: " + matrix[1][1]); // 5 System.out.println("Соңғы элемент: " + matrix[2][2]); // 9 // 4. Элементтерді өзгерту matrix[0][0] = 100; // Бірінші элементті өзгерту subjects[1][0] = "Химия"; // Пәнді өзгерту // 5. Бос 2D массив құру double[][] prices = new double[3][4]; prices[0][0] = 99.99; prices[1][2] = 149.50; // 6. Барлық элементтерді шығару (Nested loops) System.out.println("Барлық элементтер:"); for (int i = 0; i < matrix.length; i++) { for (int j = 0; j < matrix[i].length; j++) { System.out.print(matrix[i][j] + " "); } System.out.println(); // Жаңа жолға көшу }
