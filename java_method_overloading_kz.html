🔢 Overloading түрлері

1️⃣ Параметр саны бойынша

Бірдей типті, бірақ әртүрлі санда параметрлер

public void method() { }
public void method(int a) { }
public void method(int a, int b) { }

Мысал: Калькулятордағы қосу операциясы

2️⃣ Параметр типі бойынша

Бірдей саны, бірақ әртүрлі типтегі параметрлер

public void print(int value) { }
public void print(String value) { }
public void print(boolean value) { }

Мысал: Әртүрлі типтерді шығару

3️⃣ Параметр реті бойынша

Бірдей параметрлер, бірақ әртүрлі ретпен

public void method(int a, String b) { }
public void method(String a, int b) { }

Мысал: Деректерді әртүрлі ретпен өңдеу

4️⃣ Аралас overloading

Бірнеше ереже бір мезгілде қолданылады

public int calculate(int a) { }
public double calculate(double a, double b) { }
public String calculate(String a, int b, boolean c) { }

Мысал: Әртүрлі есептеу әдістері

💻 Параметр саны бойынша Overloading

🧮 Калькулятор - параметр саны бойынша overloading
public class Calculator { // 1. Параметрсіз - тек хабарлама public static void add() { System.out.println("Қосу операциясы орындауға дайын"); } // 2. Бір параметр - санды екі еседеу public static int add(int a) { return a + a; // Санды өзіне қосу } // 3. Екі параметр - стандартты қосу public static int add(int a, int b) { return a + b; } // 4. Үш параметр public static int add(int a, int b, int c) { return a + b + c; } // 5. Төрт параметр public static int add(int a, int b, int c, int d) { return a + b + c + d; } // 6. Varargs - кез келген санда параметр public static int add(int... numbers) { int sum = 0; for (int num : numbers) { sum += num; } return sum; } public static void main(String[] args) { // Әртүрлі overloaded методтарды шақыру add(); // void метод int result1 = add(5); // 5 + 5 = 10 int result2 = add(3, 7); // 3 + 7 = 10 int result3 = add(1, 2, 3); // 1 + 2 + 3 = 6 int result4 = add(1, 2, 3, 4); // 1 + 2 + 3 + 4 = 10 int result5 = add(1, 2, 3, 4, 5, 6); // varargs: 21 System.out.println("Нәтижелер: " + result1 + ", " + result2 + ", " + result3 + ", " + result4 + ", " + result5); } }

💡 Маңызды ескерту:

Varargs (...) параметрі бар метод әрқашан соңғы болып таңдалады. Егер add(1, 2, 3) шақырсаңыз, алдымен add(int, int, int) ізделеді, одан кейін add(int...) қаралады.
